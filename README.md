**Multi Ping utility**

- ping multiple addresses, either IP or FQDN and display results to screen

- user provides list of addresses either manually, one at a time, or in a file "ip_list.txt" with one address per line
- 20 address limit
- user presses 'esc' to end program

- diplay shows columns of address, ping count/miss, percentage, latency
-   green row if ping succeeds, red row if ping failed
